asn1oid Type for PostgreSQL
===========================

The asn1oid extension provides a datatype for ASN.1 OIDs in PostgreSQL.

```
CREATE EXTENSION asn1oid;

# SELECT '1.3.6.1.4.1'::asn1oid;
   asn1oid
─────────────
 1.3.6.1.4.1
```
